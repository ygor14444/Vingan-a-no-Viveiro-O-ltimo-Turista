## Vingança no Viveiro O Último Turista

"Vingança no Viveiro" é um jogo de sobrevivência top-down 2D onde o jogador controla um turista comum que se vê preso num viveiro macabro. O objetivo é sobreviver a hordas de uvas mutantes carnívoras usando uma escopeta que evolui e fica mais destrutiva à medida que o jogador derrota os inimigos e absorve a energia deles. Após eliminar uma quantidade específica de monstros, o verdadeiro Chefão desperta para o combate final.

Realizado durante a aula Topicos especiais em técnologia.


Feito por Ygor Dhener Miranda Barbosa e Geraldo Souza de Oliveira
